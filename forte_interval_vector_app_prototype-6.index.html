Post-tonal prototype Prime Form + Interval Vector Static HTML版

Forte / Interval Vector Explorer

ピッチクラス集合を入力すると、normal order・prime form・interval vector・簡易キャラクター指標・PCクロック図を同時に可視化します。
この版は 1ファイルだけ で動くので、GitHub Pages / Netlify / Vercel に置けば URLを開くだけで使えるアプリ にできます。

入力

入力した pitch classes を C4 基準で再生します。和音再生は同時発音、アルペジオ再生は低い順に順次発音です。初回はブラウザの音声許可のため、再生ボタンを一度押してください。

集合情報

URL公開の最短手順

1. このファイルを index.html という名前で保存
2. GitHub に新しいリポジトリを作る
3. index.html をアップロードする
4. GitHub Pages を ON にする
5. 発行された URL を開けば完成
Netlify Drop に index.html をドラッグしても、すぐURL化できます。
